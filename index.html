Welcome to exacto float!

A website that allows you to find trade up contracts that will result in a skin with a very specific float.

How it works

We try to make trading up for skins with specific floats as easy as possible. You only need to let us know what skin you would like and what float value you would like the skin to have. We will try to find a contract that will result in that exact skin, we will also give you the cheapest combination if multiple contracts where found.
